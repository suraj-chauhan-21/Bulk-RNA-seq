### 3. **Analysis Scripts**

#### fastqc_analysis.sh
- **Type:** Bash script
- **Purpose:** Quality control of raw sequencing data
- **Steps:**
  1. Runs FastQC on all FASTQ files
  2. Aggregates results with MultiQC
  3. Generates HTML report
  4. Provides quality summary statistics
- **Output:** FastQC reports, MultiQC report
- **Time:** ~2-10 min per 100M reads

#### trimming.sh
- **Type:** Bash script
- **Purpose:** Read processing - adapter and quality trimming
- **Tool:** Trimmomatic
- **Features:**
  - Paired-end read processing
  - Adapter removal
  - Quality filtering
  - Length filtering
  - Per-sample statistics
- **Output:** Trimmed FASTQ files, logs, metrics
- **Time:** ~5-20 min per sample

#### alignment.sh
- **Type:** Bash script
- **Purpose:** Read alignment to reference genome
- **Tool:** HISAT2
- **Features:**
  - SAM to BAM conversion
  - BAM sorting and indexing
  - Alignment statistics (flagstat)
  - Comprehensive logging
  - Parallel processing support
- **Output:** Sorted BAM files, indices, metrics
- **Time:** ~10-30 min per sample

#### quantification.sh
- **Type:** Bash script
- **Purpose:** Gene-level read counting
- **Tool:** featureCounts
- **Features:**
  - Count matrix generation
  - Per-sample statistics
  - Count summary extraction
  - Matrix cleaning for R analysis
- **Output:** Count matrix, assignment statistics
- **Time:** ~5-15 min for all samples

#### dge_analysis.R
- **Type:** R script
- **Purpose:** Differential gene expression analysis
- **Tool:** DESeq2
- **Features:**
  - Sample loading and QC
  - Normalization (DESeq2 method)
  - Differential expression testing
  - Results filtering and sorting
  - Multiple visualization types:
    - PCA plot
    - MA plot
    - Volcano plot
    - Gene heatmap
    - Expression boxplots
  - Statistical summaries
- **Output:** CSV results, 5+ PDF plots
- **Time:** ~5-10 min

#### enrichment_analysis.R
- **Type:** R script
- **Purpose:** Functional interpretation of differentially expressed genes
- **Tools:** clusterProfiler
- **Analysis:**
  - Gene Ontology (GO) enrichment (BP, MF, CC)
  - KEGG pathway analysis
  - Upregulated vs downregulated comparison
  - Gene ID conversion (Symbol → Entrez)
- **Visualizations:**
  - Dotplots
  - Barplots
  - Comparative plots
- **Output:** CSV enrichment tables, 6+ PDF plots
- **Time:** ~10-20 min

## File Organization & Usage

### Installation Phase
```
1. Clone repository
2. bash setup.sh          # Run setup.sh
3. conda activate bulk_env
4. source .env           # Load environment variables
```

### Pre-Analysis Phase
```
5. Prepare sample_metadata.csv
6. Download FASTQ files
7. Edit config.yaml if needed
```

### Analysis Phase (Sequential)
```
8. bash $SCRIPTS/fastqc_analysis.sh
9. bash $SCRIPTS/trimming.sh
10. bash $SCRIPTS/alignment.sh
11. bash $SCRIPTS/quantification.sh
12. Rscript $SCRIPTS/dge_analysis.R
13. Rscript $SCRIPTS/enrichment_analysis.R
```
